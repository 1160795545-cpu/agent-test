# 智能体体验入口 Demo

一个用于展示多模态智能体能力的前端 Demo 页面，适合做方案试用、能力演示和原型讲解。

## 项目简介

这个项目提供了一个可直接打开使用的静态网页，围绕 3 个能力入口组织体验流程：

- 事件判断
- 图像描述
- 视频摘要

页面支持上传图片或视频、输入规则或关注方向，并在前端生成演示级分析结果，适合在没有后端服务的情况下快速展示交互流程。

## 功能说明

### 1. 事件判断

上传一张图片并输入判断规则，点击运行后展示：

- 和判断规则相关的图像描述
- 是否发生目标事件
- 简要分析依据

### 2. 图像描述

上传一张图片并输入关注方向，点击运行后展示：

- 面向关注方向的图像描述结果

### 3. 视频摘要

上传一段视频并输入关注方向，点击运行后展示：

- 关键帧截图
- 每张关键帧的描述
- 基于关键帧内容生成的整段视频摘要

## 技术方式

- 纯前端静态页面
- `HTML + CSS + JavaScript`
- 不依赖后端服务
- 可直接本地打开 `index.html` 体验

## 使用方式

### 本地直接打开

直接双击 `index.html`，或在浏览器中打开该文件即可。

### 本地开发

如果你希望通过本地服务查看，也可以在项目目录中启动一个静态文件服务，例如：

```bash
python3 -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```
